# SSC 2026 Submission Checklist — Unseen

## 1) 필수 기능 확인 (실기기)
- [ ] 앱 실행 시 카메라 권한 팝업 정상 노출
- [ ] 권한 허용 시 실시간 카메라 피드 표시
- [ ] 권한 거부 시 샘플 모드로 자동 폴백
- [ ] Vision mode 토글(4종) 전환 시 화면 즉시 변경
- [ ] 텍스트 인식 후 PASS/FAIL 배지 오버레이 표시
- [ ] FAIL 항목이 실제 저대비 텍스트에서 재현됨
- [ ] 카메라 화면 탭 시 색상 상세 시트 표시
- [ ] 대체 색상 제안(HEX) 표시

## 2) 데모 안정성
- [ ] 조명 밝은 환경/어두운 환경 각각 테스트
- [ ] 인쇄물 + 모니터 화면 각각 테스트
- [ ] 텍스트가 많은 화면에서 프레임 저하 없는지 확인
- [ ] 앱 재실행 시 크래시 없음

## 3) 접근성
- [ ] Dynamic Type에서 레이아웃 깨짐 없음
- [ ] VoiceOver로 주요 버튼/토글/상태 읽힘
- [ ] 색 정보 외 텍스트 상태(PASS/FAIL)가 함께 전달됨

## 4) 제출 패키지
- [ ] `unseen.swiftpm` 폴더만 제출 대상인지 최종 확인
- [ ] `Supporting/Info.plist`의 카메라 권한 문구 확인
- [ ] 외부 네트워크/비밀키 의존 없음 확인 (offline)
- [ ] README 최신 상태 반영

## 5) 심사 포인트(짧게 말하기)
- 카메라가 없으면 존재할 수 없는 앱
- 실시간 시뮬레이션 + 자동 진단 + 대체 색상 제안
- 체험형이 아니라 반복 사용 가능한 검수 도구
