# Unseen 3분 데모 스크립트 (한국어)

## 0:00 ~ 0:15
"Unseen은 iPad 카메라로 실제 세계를 비추면, 색각이상 시야와 텍스트 대비 문제를 실시간으로 보여주는 앱입니다."

## 0:15 ~ 0:50
- 카메라로 포스터/교재를 비춤
- Vision mode를 Normal → Deuteranopia로 전환

"지금 보이는 것처럼 빨강-초록이 구분이 어려워집니다. 이게 실제 사용자의 시야입니다."

## 0:50 ~ 1:40
- 텍스트 인식 결과 PASS/FAIL 배지 표시 보여주기

"앱이 텍스트를 자동 인식해서 WCAG 대비 기준으로 PASS/FAIL을 즉시 계산합니다."

## 1:40 ~ 2:20
- FAIL 영역 근처를 탭
- 색상 상세 시트(HEX/RGB, 모드별 변환, 대체 색상) 확인

"문제를 보여주는 데서 끝나지 않고, 바로 고칠 수 있는 대체 색상까지 제안합니다."

## 2:20 ~ 3:00
"핵심은 ‘왜 앱이어야 하는가’입니다. 이건 파일 필터가 아니라 실물 환경을 실시간 진단하는 도구입니다. 디자이너와 교사가 반복해서 쓰는 검수 앱입니다."
